# Rincolabs 🐧

**Linux-first open source software, built with care.**

Rincolabs is a small open source initiative focused on building thoughtful desktop tools for Linux and other platforms. The goal is simple: create software that is useful, honest, maintainable, and able to grow over time.

We are not trying to promise everything at once. We are building step by step, learning from real use, and improving with patience. 🌱

## What we care about

* **Linux as a first-class platform** 🐧
  Our work starts with Linux, not as an afterthought, but as a serious environment for creators, developers, and professionals.

* **Open source development** 🛠️
  We believe in transparent work, community trust, and long-term access to the tools people depend on.

* **Sustainable progress** ☕
  Good software takes time. We focus on steady development, careful decisions, documentation, testing, packaging, and maintenance.

* **Practical tools for real workflows** ✨
  We prefer useful, grounded features over hype. The goal is to build tools that feel good to use in everyday work.

## Projects

### Hazor Studio

Hazor Studio is an open source image editor currently in public beta. It is being built with care, one step at a time, with a focus on a solid editing experience for desktop creators.

The project already includes core editing systems such as layers, brushes, selections, transforms, masks, and experimental AI-assisted workflows. It is still evolving, and feedback from real use helps guide the next improvements.

👉 [Learn more about Hazor Studio](https://rincolabs.org/projects/hazor-studio/)
👉 [View the source code](https://github.com/rincolabs/hazor-studio)

## Why Rincolabs exists

Linux deserves high-quality creative and professional software that feels native, respectful, and sustainable.

Rincolabs exists to contribute to that ecosystem with open source projects that can grow carefully over time. There is a long road ahead, but every release, fix, test, and contribution helps move the work forward. 🚀
